# Haiku feature

## Generates a poem from the chosen channels etc. in the users connected slack workspace

### Clicking the genereate button reveals a modal with a form. The generated Haiku will be presented in the tile to the right. Here you can choose if you want to save the haiku or try again.

### If you want to save it, an input is revealed and a save button. The input should contain the title of the haiku before you save.
